Session Management

User session handling with activation, deactivation, and history tracking

Version: 1.2.0 Status: Active Run Tests →

Overview

The Session Management feature provides comprehensive user session handling capabilities for the Agent Hub API. It ensures that only one session per user can be active at a time, maintains session history, and provides detailed session analytics.

API Endpoints

POST /session/activate

Activate a new session, deactivating any existing active session

POST /session/deactivate

Deactivate the current active session

GET /session/active

Get the currently active session details

GET /session/history

Retrieve session history with pagination

Key Features

  • ✓ Single active session enforcement per user
  • ✓ Automatic deactivation of previous sessions
  • ✓ Complete session history tracking
  • ✓ Session analytics and reporting
  • ✓ JWT authentication support
